## âœ¨ Features

This website is a 3-page static site with a feature-rich homepage.

* **Fully Responsive Design:** Adapts beautifully to all screen sizes, from mobile phones to desktops.
* **Multi-Page Structure:**
    * `Home`: A landing page with multiple engaging sections.
    * `About Us`: A clean, two-column layout.
    * `Contact Us`: A modern form with an info sidebar.
* **Modern CSS:**
    * Layouts built with **CSS Flexbox** and **Grid**.
    * Custom CSS variables for a consistent and easy-to-manage color palette.
    * Clean, reusable **card components** for destinations.
    * Subtle hover effects and box-shadows for a modern feel.
* **JavaScript Functionality:**
    * **Responsive "Hamburger" Menu:** A mobile-friendly navigation that toggles on click.
    * **Form Validation:** Client-side validation for the "Contact Us" form to ensure all fields are filled.

### Home Page Sections:
* **Hero:** A full-screen welcome image with a strong call-to-action.
* **Featured Destinations:** A 3-column card grid.
* **How It Works:** A 3-step feature section.
* **Testimonials:** A "What Our Travelers Say" section with a blockquote.
* **Why Choose Us:** An icon-based feature grid highlighting key benefits.
